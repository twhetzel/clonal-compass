Clonal Compass

A grounded interpretation layer for single-cell immune repertoire analysis.

Clonal Compass helps researchers interpret paired scRNA-seq and TCR-seq analysis outputs by combining computed clone expansion, cluster evidence, marker summaries, and epitope database matches into reviewable reports and a grounded chat interface.

Clonal Compass is intended for exploratory research support only. It reports computed evidence and flags uncertainty; it does not make diagnostic, therapeutic, or patient-level claims.
